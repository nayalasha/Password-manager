# 🔐 Password Manager

A desktop Password Manager built with Python and CustomTkinter.

This application allows users to securely save website login information using a SQLite database.

---

## Features

✅ Save passwords

✅ Generate secure passwords

✅ Store data using SQLite

✅ Display saved passwords

✅ Hidden password field

✅ Dark mode interface

✅ Pop-up notifications

---

## Built With

- Python
- CustomTkinter
- SQLite3
- Tkinter Messagebox

## Installation

Clone the repository:

```bash
git clone https://github.com/nayalasha/Password-manager.git
```

Go into the project folder:

```bash
cd Password-manager
```

Install CustomTkinter:

```bash
pip install customtkinter
```

Run the application:

```bash
python main.py
```

---

## How It Works

1. Enter a website.
2. Enter a username.
3. Enter a password.
4. Click "Save Password."
5. The information is stored in a SQLite database.
6. Saved passwords appear inside the application.
